## ⚠️ 我们部分实现的功能

### 1. 高光模式

**参考代码**：
```hlsl
// 可在硬高光和软高光之间切换
if (_Is_SpecularToHighColor)
    specular = pow(NdotH, _HighColor_Power);  // 软高光
else
    specular = step(1.0 - _Tweak_HighColorMaskLevel, NdotH);  // 硬高光
```

**我们的实现**：
```hlsl
// 只有硬高光
float specularMask = step(1.0 - _SpecularSize, NdotH);
```

**改进建议**（可选）：
- L2时添加软高光模式开关
- 或报告中说明"采用硬高光更符合三渲二风格"

### 2. 相机距离补偿

**参考代码**：
```hlsl
// 完整的FOV补偿和正交相机支持
float fov = atan(1.0 / unity_CameraProjection._m11) * 2.0;
float cameraMulFix = distance * fov / 60.0;

// 正交相机特殊处理
if (unity_OrthoParams.w)
    cameraMulFix = orthoSize * 125;
```

**我们的实现**：
```hlsl
// 简化的距离缩放
float distanceScale = saturate(distance * 0.2);
```

**影响**：
- 小场景：影响不大
- 大场景或变化FOV：可能需要调整

**改进建议**（可选）：
- 如果测试中发现问题，再添加FOV补偿
- 当前版本对课程项目足够
