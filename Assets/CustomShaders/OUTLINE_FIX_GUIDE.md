# 描边断裂修复方案

## 问题分析

**法线外扩描边为什么会断裂？**

法线外扩描边在以下情况容易出现断裂：

1. **硬边（Hard Edges）**
   - 模型的法线在边缘处不连续
   - 相邻顶点有不同的法线方向
   - 导致描边向不同方向扩展，产生间隙

2. **尖锐角度**
   - 如牛的角、蹄子等尖锐部位
   - 法线方向差异极大
   - 描边容易分离

3. **UV接缝处**
   - 建模软件在UV展开处分裂顶点
   - 虽然位置相同，但法线可能不同
   - 造成描边断裂

---

## 解决方案

### 方案1：混合描边（Hybrid Outline）⭐ 推荐

**原理**：
结合法线方向和位置方向，自动在两者之间找到最佳平衡点。

**优点**：
- ✅ 自动适应不同模型
- ✅ 减少硬边断裂
- ✅ 保持描边连续性
- ✅ 无需额外设置

**实现**：
```hlsl
float3 GetHybridDirection(float3 positionOS, float3 normalOS)
{
    float3 posDir = normalize(positionOS);
    float3 normal = normalize(normalOS);
    
    // 根据法线与位置的对齐程度加权
    float weight = abs(dot(normal, posDir));
    return normalize(lerp(normal, posDir, weight * 0.5));
}
```

**使用场景**：
- 复杂角色模型（如你的牛模型）
- 有硬边的机械模型
- 大多数情况下的默认选择

---

### 方案2：位置扩展（Position-based）

**原理**：
沿着顶点位置方向（从物体中心向外）扩展，而不是沿法线。

**优点**：
- ✅ 对称物体效果好
- ✅ 避免法线不连续问题
- ✅ 硬边不会断裂

**缺点**：
- ❌ 非中心对称物体可能失真
- ❌ 细长部位描边可能不均匀

**使用场景**：
- 球体、立方体等对称物体
- 中心对称的角色（正面站姿）

---

### 方案3：纯法线（Normal-based）

**原理**：
保持原始的法线外扩方法（原Shader的方式）。

**优点**：
- ✅ 性能最好
- ✅ 平滑表面效果完美
- ✅ 实现简单

**缺点**：
- ❌ 硬边会断裂
- ❌ 需要高质量法线

**使用场景**：
- 平滑的有机体模型
- 法线质量好的模型
- 性能敏感的情况

---

## 新增参数说明

### Outline Mode（描边模式）

在Inspector中选择描边方法：

- **Normal**: 纯法线外扩（原方法）
- **Position**: 位置扩展
- **Hybrid**: 混合方法（推荐，默认）⭐

### Outline Fix Options（描边修复选项）

**Z Offset**（Z偏移）
- 范围：-1 到 1
- 默认：0
- 作用：防止描边与模型表面Z-fighting
- 调整：如果描边闪烁，设置为0.1-0.5

**Min Width**（最小宽度）
- 范围：0 到 0.01
- 默认：0.0001
- 作用：防止远处描边消失
- 调整：如果远处描边太细，增大到0.001

**Max Width**（最大宽度）
- 范围：0 到 0.05
- 默认：0.02
- 作用：防止近处描边过粗
- 调整：如果近处描边太粗，减小到0.01

## 技术原理详解（报告用）

### Hybrid混合算法

```hlsl
// 步骤1：获取两个候选方向
float3 normalDir = normalize(normalOS);      // 法线方向
float3 positionDir = normalize(positionOS);  // 位置方向

// 步骤2：计算对齐程度
// 如果法线和位置方向接近，说明表面是圆滑的
// 如果差异大，说明表面有硬边
float alignment = abs(dot(normalDir, positionDir));

// 步骤3：自适应混合
// alignment = 1 (对齐): 更多使用位置方向（避免硬边问题）
// alignment = 0 (垂直): 更多使用法线方向（保持正常描边）
float3 finalDir = normalize(lerp(normalDir, positionDir, alignment * 0.5));
```

**为什么有效**：
- 在平滑区域：法线方向占主导，描边跟随表面形状
- 在硬边区域：位置方向增加权重，减少断裂
- 自动平衡：无需手动调整

如果Hybrid模式仍有问题，可以尝试：

### 方法1：法线平滑（需要建模软件）

在Blender/Maya中：
1. 选中模型
2. 右键 → Shade Smooth
3. 在尖锐边缘处手动标记硬边
4. 重新导出

### 方法2：使用顶点颜色控制

修改Shader，使用顶点颜色控制描边宽度：
```hlsl
// 在描边顶点着色器中
float widthControl = input.color.r; // 红通道控制宽度
finalWidth *= widthControl;
```

在建模软件中：
- 问题区域的顶点颜色设为深红（加粗描边）
- 正常区域保持白色

### 方法3：描边宽度贴图

使用贴图精确控制每个部位的描边：
```
在Photoshop中绘制：
- 白色：正常描边
- 黑色：无描边
- 灰色：细描边
- 需要加粗的部位：更白

保存为OutlineWidthMap并指定给材质
```

## 常见问题

### Q1: 为什么我的模型改用Hybrid后反而效果差了？
**A**: 可能是模型本身法线质量很好，纯Normal模式已经足够。尝试调整混合权重或使用Normal模式。

### Q2: Hybrid模式描边粗细不均匀
**A**: 
1. 调整 Outline Width 到较小值（0.004-0.006）
2. 尝试Position模式
3. 使用OutlineWidthMap精确控制

### Q3: 修改Outline Mode后没有效果
**A**: 
1. 保存Shader文件
2. 等待Unity重新编译
3. 重新应用材质到模型
4. 确认Inspector中Material使用了新Shader

### Q4: 性能下降
**A**: Hybrid模式的额外开销极小。如果性能下降，检查：
1. 是否误开了大量Debug窗口
2. 场景中物体是否过多
3. 其他性能瓶颈
